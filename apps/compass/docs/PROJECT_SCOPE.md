# Project Scope

NightRaven Compass is a visual project-guidance app for a non-coder builder using NightRaven and NightRaven.

The app must answer one question first:

```text
What is the next correct step?
```

## Phase 1 Scope

- React + TypeScript + Vite app.
- Mock data only.
- App shell and sidebar.
- Dashboard with project, phase, next action, progress, blocker, decision, Not Now, and recommended prompt cards.
- Future areas appear only as navigation placeholders.

## Not Now

- Cloud sync.
- Real AI automation.
- Repo auto-editing.
- Plugin marketplace.
- MCP manager.
- Database.
- Mobile app.
