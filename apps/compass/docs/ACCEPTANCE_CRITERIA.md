# Acceptance Criteria

MVP Phase 1 is complete when:

- App opens successfully.
- Dashboard shows project name.
- Dashboard shows project concept.
- Dashboard shows current phase.
- Dashboard shows next best action.
- Dashboard shows progress types.
- Dashboard shows blockers.
- Dashboard shows decisions.
- Dashboard shows Not Now list.
- Dashboard shows recommended prompt.
- Mock data is connected.
- Sidebar exists.
- Future pages have placeholder navigation.
- No real AI automation exists.
- No cloud sync exists.
- No repo auto-editing exists.
- README explains how to run the app.
- Auditor can verify Phase 1.
