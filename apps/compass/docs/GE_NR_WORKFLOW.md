# GE / NR Workflow

## Roles

- God's Eye thinks: scope, intent, risk, phase order, and memory.
- NightRaven Builder builds: scoped implementation only.
- Auditor verifies: acceptance criteria, scope creep, and readiness.
- Compass points: next action, blocker, prompt, and Not Now boundary.

## Phase 1 Flow

1. Compass displays the current project and phase.
2. Compass identifies the highest priority Now task.
3. Compass shows what is blocked or delayed.
4. Compass gives the next prompt to send.
5. Auditor checks before a task becomes Done.

## Rule

No task is Done until Auditor passes it.
